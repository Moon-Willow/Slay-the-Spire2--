# Slay the Spire2-明月栞那

杀戮尖塔2mod，亡灵契约师皮肤-明月栞那

该mod使用B站大佬 比逗逗丶提供的模版制作，主页：https://space.bilibili.com/40726676

使用方法：下载Akizuki\_kanna.zip，将其解压在游戏目录 mods文件夹中，在游戏中加载该mod即可。

说明：杀戮尖塔2 安装mod和不安装mod是两个存档，如果您是第一次安装mod，那么会是一个新的存档，关于安装mod后如何使用原来的存档，请参考网上相关教程。

一些必要的说明：

1\.本mod的角色动画使用的是序列帧动画，所以一些情况下可能会不和谐（比如短时间内释放多次动画，可能只播放一次，又或者当前动画被打断播放另一个动画），这些是正常的。

2\.在进入游戏后的第一场战斗时由于要加载资源，所以加载时间会长一些（一般几秒），后续的加载时间就正常了。

3\.关于奥斯提（皮肤为御帝），目前只有攻击、受击和待机动画，至于其他动画（如死亡、复活）本来是做了，但是由于一些bug，所以删掉了。

4\.多人联机中角色手部没有替换，因为AI抽卡抽不出像样的图片。

5\.多人联机中篝火处，任何需要选中的操作（比如愈合），该mod的角色不能指向角色，所以如果您需要执行此类操作，可以用图示的方法。

6\.目前提交的已经是作者尽最大可能做的最好的版本了，而且作者也是用大佬的模板做的mod，不会写代码，所以如果有bug的话，作者不一定能解决。
